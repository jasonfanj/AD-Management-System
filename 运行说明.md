# 运行说明

## 方式一：使用启动脚本（推荐）

### Windows用户

1. **双击运行 `start.bat`**
   - 脚本会自动检查Python环境
   - 自动创建虚拟环境（如果不存在）
   - 自动安装依赖包（如果未安装）
   - 自动启动后端服务

2. **等待服务启动**
   - 看到 "API地址: http://localhost:5000/api" 表示启动成功

3. **打开前端界面**
   - 在文件管理器中找到 `frontend/index.html`
   - 双击打开（或右键选择浏览器打开）

### Linux/Mac用户

```bash
chmod +x start.sh
./start.sh
```

## 方式二：手动启动

### 1. 检查Python环境

```bash
python --version
# 或
python3 --version
```

需要Python 3.8或更高版本。

### 2. 创建虚拟环境（首次运行）

```bash
python -m venv venv
```

### 3. 激活虚拟环境

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/Mac:**
```bash
source venv/bin/activate
```

### 4. 安装依赖

```bash
pip install -r requirements.txt
```

### 5. 启动服务

```bash
python app.py
```

### 6. 打开前端

- 浏览器打开 `frontend/index.html` 文件

## 常见问题

### Q: 提示"未检测到Python"？

A: 请先安装Python 3.8+：
- 下载地址: https://www.python.org/downloads/
- 安装时勾选"Add Python to PATH"

### Q: 端口5000被占用？

A: 修改 `config.py` 中的 `PORT` 设置，或设置环境变量：
```bash
set PORT=8080
```

### Q: 前端无法连接后端？

A: 确保：
1. 后端服务已启动
2. 浏览器控制台没有CORS错误
3. API地址配置正确（默认: http://localhost:5000/api）
